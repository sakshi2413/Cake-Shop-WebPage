Available Endpoints

GET /api/health
Check if the API server is running and responding correctly.
GET /api/products
Retrieve all products from the database (cakes, pastries, new arrivals).
GET /api/products/category/:category
Get products filtered by category. Categories: cake, pastry, new-arrival
GET /api/products/:id
Get details of a specific product by its ID.
GET /api/orders
Retrieve all orders from the database with customer details and order items.
GET /api/orders/:id
Get details of a specific order including all order items.
POST /api/orders
Create a new order. Requires customer details and order items in request body.
PATCH /api/orders/:id/status
Update the status of an order. Status options: pending, processing, delivered, cancelled

📌 How to Use This API

Base URL: http://localhost:3000/api

Authentication: None (for development)

Response Format: JSON

CORS: Enabled for localhost
